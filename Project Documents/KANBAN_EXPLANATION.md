Kanban Board Explanation

What is a Kanban Board?
--------------------------

A Kanban board is a visual project management tool used to track tasks as they move through different stages of development. It uses columns to represent workflow stages such as "To Do," "In Progress," and "Done."

How the Board Visualizes Workflow
----------------------------------

The board is structured into columns:

To Do: Tasks that are not started
In Progress: Tasks currently being worked on
Testing: Tasks being verified
Done: Completed tasks

Each task moves from left to right, showing its current status.

Work-In-Progress (WIP) Limits
------------------------------

To prevent overload, the board limits the number of tasks in progress:

Maximum of 3 tasks in "In Progress"

This ensures:
Better focus
Faster completion
Reduced bottlenecks

Agile Support
--------------

The Kanban board supports Agile principles by:

Allowing continuous updates
Supporting iterative development
Providing visibility into progress
Enabling flexibility to adapt to changes

It helps ensure that tasks are completed efficiently while maintaining system quality.
