Smart Student Assignment Tracker

Project Overview
The Smart Student Assignment Tracker is a web-based system that enables students to effectively manage academic assignments, deadlines, and coursework. Many students struggle to keep track of several assignments from different modules, which can result in missed deadlines and poor academic planning. This system provides a consolidated platform for students to arrange assignments, track progress, and receive reminders about upcoming tasks. The purpose of this system is to improve student productivity and reduce academic stress by providing a simple and structured digital solution for assignment management.

Key Features
•	User registration and authentication
•	Module or course management
•	Assignment creation and tracking
•	Assignment deadlines and priority settings
•	Dashboard displaying upcoming assignments
•	Assignment progress tracking
•	Reminder notifications for deadlines

System Purpose
The system enables students to keep a clear picture of their academic workload. Students can better organize their studies and manage their time by centralizing assignments, deadlines, and progress.
